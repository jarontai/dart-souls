# Dart Souls - 《Dart 之魂》

《Dart 之魂》，即 Dart 编程要点 \(编写中......\)

## 简介

《Dart 之魂》是一本关于Dart编程的开源小书，目标是用简单的文字归纳 Dart 的主要知识点。

## 定位

本书面向的读者是具备一定编程经验，希望快速上手 Dart 的开发者。本书定位为入门级，如果想深入学习 Dart，你还需要查阅 Dart 官网或其他进阶书籍。
