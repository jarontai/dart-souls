# The Little Dart Book -《Dart简易教程》

一本介绍Dart的小书 \(编写中...\)

### 简介

目标：用简单的文字归纳Dart的主要知识点

面向读者：有其他编程语言经验，希望快速上手Dart的开发者

注意：本书定位为入门型笔记，想更深入地学习Dart，你还需要查阅[Dart官网](https://www.dartlang.org/)或其他进阶书籍
